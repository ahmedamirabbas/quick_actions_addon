# Changelog

All notable changes to Quick Actions Panel will be documented in this file.

## [1.0.0] - 2025-11-29

### Added
- Screenshot tool with automatic UI exclusion (F9)
- Scene reload functionality (Ctrl+R)
- FPS counter with color-coded display (F3)
- Timescale control slider (0.1x to 3.0x)
- Timescale hotkeys (Ctrl+/-, Ctrl+0)
- Draggable panel with mouse
- Resizable panel (200x250 to 400x600)
- Minimize/maximize button
- Panel toggle hotkey (Ctrl+Shift+Q)
- Editor-only mode for development
- Runtime mode with CanvasLayer overlay
- Comprehensive keyboard shortcuts
- User data folder screenshot saving

### Technical
- Full GDScript type annotations
- CanvasLayer rendering (layer 4096)
- Autoload singleton for runtime
- Editor plugin integration
- Clean plugin enable/disable lifecycle
